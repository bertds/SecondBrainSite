Wat is het voordeel om hiervoor functies te schrijven ?

  • je kan de transformaties makkelijk ook op andere datasets gebruiken.
  • je zal langzaam een bibliotheek aan functies maken dat je kan hergebruiken
  • je kan deze functie ook gebruiken in je live systeem als er nieuwe data binnenkomt en je die wil transformeren vooraleer die te gebruiken in je algoritme.
  • het stelt je in staat om makkelijk verschillende combinaties van transformaties uit te proberen en te zien welke de beste zijn.

Ontbrekende waarden

Indien er ontbrekende waarden zijn kan je die op volgende manieren mee omgaan (pandas) :

  • rijen verwijderen : df.dropna()
  • kolom/attribuut verwijderen : df.drop()
  • ontbrekende waarden invullen met een waarde : df.fillna()
    hierbij kan je de median waarde bepalen van de training set en die invullen. Opgelet ! Wel bijhouden om ook in de testset en indien je systeem live gaat ontbrekende waarde in te vullen.

Scikit-Learn SimpleImputer vult ontbrekende waarden in. (median kan echter enkel met numerieke waarden dus hiervoor moet je df opsplitsen met/zonder numerieke kolommen)
from sklearn.impute import SimpleImputer
imputer = SimpleImputer(strategy='median')
housing_num = housing.drop('ocean_proximity', axis=1)
imputer.fit(housing_num)
x = imputer.transform(housing_num)
