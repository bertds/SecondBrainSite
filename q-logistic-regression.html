Q: Wat is een confusion matrix en hoe is die opgebouwd ?
A: Well, it is a performance measurement for machine learning classification problem where output can be two or more classes. It is a table with 4 different combinations of predicted and actual values.
It is extremely useful for measuring Recall, Precision, Specificity, Accuracy and most importantly AUC-ROC Curve.
| CM | P' (predicted)|n' (predicted)|
| --- | --- | --- |
| P (actual) | True positive | False negative |
| n (actual)| False positive | True negavtive|

Q: Wat kan je afleiden uit een confusion matrix ? Wat zijn de formules hiervoor ?
A: Accuracy, Recall, Precision, F1 score :
Accuracy = $\dfrac{TP + TN}{TP + FP + FN + TN}$
Recall = $\dfrac{TP}{TP + FN}$
Precision = $\dfrac{TP}{TP + FP}$
F1 score = $\dfrac{2 * (Recall * Precision)}{Recall + Precision}$

Q: Wat is niet gebalanceerde data ? Wat kan het effect zijn op een classifier ?
A: Het model gaat voorkeur geven aan category die vaak voorkomt

Q: Hoe kan je voorkomen dat een classifier voorkeur gaat geven aan een klasse die minder voorkomt in de data ?
A: De mogelijkheden zijn :

  • verzamel meer data van de minderheidsklasse
  • undersampling : klasses uitbalanceren (data uit meerderheidsklasse verwijderen)
    Nadeel : dat eigenschappen die in geschrapte data zaten gaan verloren
  • oversampling : minderheidsklasse vermeerderen/copieren
    Nadeel : geen nieuwe informatie toegevoegd
  • andere scoring parameters/metric kiezen : model kiezen met beste F1 score, of recall ipv accuracy
  • class weight balancing : misclassificatie van sample uit minderheidsklasse leiden in training tot hoger loss (meer afgestraft)
  • data augmentation - SMOTE : from imblearn_over_samplng import Borderline.SMOTE

Q: Wat is ROC ? en waarvoor gebruiken we het ?
A: ROC (Receiver Operating Characteristic) gebruiken we voor het evalueren van binaire classifiers. A graphical plot that illustrates the diagnostic ability of a binary classifier system as its discrimination threshold is varied.

Q: Wat is auROC ?
A: auROC (Area under ROC)

Q: Wat is TPF en FPR ikv ROC ?
A: TPR (True Positive Rate) (= recall) en FPR (False Positive Rate). Op basis TPR en FPT wordt de ROC-curve bepaald.

Q: Wat is ROC hell en ROC heaven ?
A: ROC hell is waar TPR = 0 en FPR = 1. ROC heaven is waar TPR = 1 en FPR = 0.
